# Trivia-Pro
 
![Repo Size](https://img.shields.io/github/repo-size/schwynf/Trivia-Pro) <br> 
## Description <span id="d"></span> 
Quiz builder with a random question generator.

This is a education/entertainment application. The information is saved in Local Storage. Application features:
1. Quick and eazy quiz builder
1. Random question generator
1. Timed quizzes with audio
1. Print quiz capability

## Installation <span id="i"></span> 
1. Clone this repository.
    ```
    git clone https://github.com/schwynf/Triva-Pro.git
    ```
1. Navigate into the cloned directory.
    ```
    open index.html in browser
    ```
